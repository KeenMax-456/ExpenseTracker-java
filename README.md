# SpendWise Buddy – Personal Expense Tracker

A Java Swing desktop application for tracking personal income and expenses.

## Features

* Add income and expense transactions
* Categorize transactions (Food, Travel, Bills, etc.)
* Dashboard showing total income, expenses, and balance
* Transaction table
* Delete transactions
* Clean Java Swing UI

## Technologies Used

* Java
* Java Swing
* ArrayList
* Object-Oriented Programming

## Project Structure

Main.java – Starts the application
Transaction.java – Transaction data model
DataManager.java – Handles calculations and storage
ExpenseTrackerUI.java – Graphical user interface

## How to Run

1. Open project in VS Code or IntelliJ
2. Run `Main.java`
3. The expense tracker window will open
